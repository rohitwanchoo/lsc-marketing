FAQ

Common questions

Do I need Google Search Console or any API keys to start?
No. The only required key is your OpenAI API key — everything else is optional. GSC, HubSpot, LinkedIn, SendGrid and GA4 integrations are available but not required. The platform runs fully on its own database from day one.
How much does it actually cost to run?
Each AI job costs approximately $0.001 using GPT-4o-mini. The full autonomous loop (all scheduled agents running across a week) costs roughly $0.03–0.10/day. There's a configurable weekly AI cost cap ($100 default) as a safety guardrail.
Is it really autonomous? What does "autonomous" mean here?
Yes. Once configured, the cron scheduler runs all agents on their schedule without any human input — keyword discovery every Wednesday, LinkedIn strategy every Tuesday, weekly review every Monday at 5am, and so on. You can also trigger any agent manually from the Products or Agents pages.
How does product-aware keyword discovery work?
When you add a product URL, the platform fetches and analyses the page using GPT-4o-mini, extracting your product name, ICP, value proposition, features, competitors, pain points and brand tone. When you then trigger keyword discovery from that product, all of this context is injected into the prompt — so you get keywords like "Gong.io alternative for call centers" instead of generic "marketing automation software".
When should I switch from organic to paid ads?
The Revenue Orchestrator monitors three unlock thresholds automatically: 50+ organic leads/month, 3%+ conversion rate, and CAC below LTV. When all three are met, the weekly review will set paid_unlock_status.unlocked = true and explain why. Until then, stay organic — paid without proof-of-conversion just burns budget.
Can I run multiple products on one platform?
Yes. Add as many products as you need via the Products page. Each product has its own keyword set, content, leads and agent runs. The database tracks product_id across all tables so each product's performance is reported independently.
